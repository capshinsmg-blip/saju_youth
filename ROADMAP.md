# 실행 로드맵

총 기간: 준비~정식 런칭 **6~8주**, 안정 운영 진입 **3개월차**.

## 3대 병목 (이게 안 풀리면 나머지가 멈춤)
1. **사주 분석지 자동화 툴** — 20분 풀이를 무엇으로? (만세력 자동분석 → 템플릿 삽입)
2. **잎사귀(진행자) 1명 양성** — 풀이 + 팔찌 + 상담연계 멘트
3. **개인정보·광고문자 수신동의** — 폼 동의 항목 필수 (미비 시 과태료 리스크)

---

## Phase 0 — 기반 세팅 (1~2주차)
| 항목 | 할 일 |
|---|---|
| 사업자 등록 | 간이과세자(연 5천만원 이하), 홈택스 |
| 공간 확정 | 맬로우 스페이스 대여 단가/예약방식 → 원가 반영 |
| 분석지 제작 | 1p(공개)·2p(빈칸) 템플릿 → `docs/02_리포트_템플릿.md` |
| 풀이 툴 | 만세력 + 표준 해석 스크립트 → `docs/01_사주분석_매칭표.md` |
| 자산 제작 | 오행카드, 포장지, 팔찌 매뉴얼 영상, QR코드 |
| 가격 검증 | 원가 견적서(원석·끈·포장·공간) |
| 결제 | 계좌이체 + 간편결제 QR |

## Phase 1 — 온라인 빌드업 (1~3주차, 병행)
- 인스타 계정 + 프로필 예약폼 링크 + 하이라이트
- 게시물 주 3~4회, **릴스 중심**
- 메타광고: 지역 타겟, 하루 3~5천원 테스트 → 반응 좋은 소재만 증액
- DM 자동응답(카드번호 → 해설지)
- ⚠️ 선팔/DM 자동·과다 활동 = 계정 정지 위험. 처음 2주는 천천히 워밍업

## Phase 2 — 오프라인 부스 파일럿 (3~4주차)
- 룰렛/추첨함 + 부스키트(QR 보드, 운세카드, 예약명부)
- 현장 멘트·동선 리허설
- 주말 1~2회 테스트 → 스캔→폼작성→예약 전환율 측정

## Phase 3 — 동시 가동 + 전환 루프 (5주차~)
- 온·오프라인 동시 운영
- 리드 CRM(구글시트): 이름/연락처/생년월일시/고민/유입경로/예약/상담연계
- 방문 → 상담 연계 KPI 추적

## Phase 4 — 최적화·확장 (3개월차~)
- 채널별 CAC 비교 → 비효율 컷
- 광고 소재 A/B, 부스 장소 로테이션
- 후기·UGC → 바이럴 환류

---

## 퍼널 KPI
`광고 도달 → 폼 작성(리드) → 예약 → 방문 → 상담 연계`
초기 핵심 KPI = **방문→상담 연계율**.

## 리스크 체크
1. 폼에 **개인정보 수집·이용 동의 + 광고성 정보 수신 동의** 필수
2. "100% 당첨·꽝없음"은 전원 증정일 때만 OK (표시광고법)
3. 결과를 금전·연애·건강으로 단정 광고 시 과장광고 소지 → "재미/무료 테스트" 톤 유지
4. 장기 전환율은 방문 경험 만족도(차·팔찌·리포트 퀄리티)에서 나옴
