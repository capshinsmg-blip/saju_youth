# 사주(만세력) 계산 원리 — 팀 학습 자료

웹 진단 페이지가 어떤 논리로 결과를 내는지 정리. 진행자(잎사귀)도 이 정도는 알아야 손님 질문에 답할 수 있음.

## 1. 사주팔자(四柱八字)란
태어난 **연·월·일·시** 4개 기둥(柱), 각 기둥은 **천간(天干) 1글자 + 지지(地支) 1글자** → 총 8글자.

- **천간(10)**: 갑乙병정무기경신임계 (甲乙丙丁戊己庚辛壬癸)
- **지지(12)**: 자축인묘진사오미신유술해 (子丑寅卯辰巳午未申酉戌亥)

### 천간·지지의 오행
| 오행 | 천간 | 지지 |
|---|---|---|
| 목(木) | 갑·을 | 인·묘 |
| 화(火) | 병·정 | 사·오 |
| 토(土) | 무·기 | 진·술·축·미 |
| 금(金) | 경·신 | 신·유 |
| 수(水) | 임·계 | 해·자 |

## 2. 4기둥 뽑는 기준 (왜 라이브러리를 쓰나)
- **연주**: 1월 1일이 아니라 **입춘(立春, ~2/4)** 기준으로 해가 바뀜
- **월주**: 달력 월이 아니라 **24절기(節氣)** 기준 (입춘=인월, 경칩=묘월 …)
- **일주**: 60갑자 연속 순환 → **기준일 정확도가 생명**
- **시주**: 2시간 단위 12지지 + 일간 기반 천간(오자둔)

→ 절기·입춘 경계는 매년 미세하게 달라서 직접 계산하면 경계일 출생자에서 오차 발생.
그래서 천문 데이터 기반 **lunar-javascript** 로 계산해 정확도를 확보함.

## 3. 오행 분포 → 부족 오행
8글자(시 모르면 6글자)의 오행 개수를 셈 → **가장 적은(또는 0개) 오행 = 보강 대상**.
보강 색상/원석은 [01_사주분석_매칭표.md](01_사주분석_매칭표.md) 사용.

## 4. 십신(十神)으로 운(運) 해석
**일간(日干, 태어난 날의 천간)** 을 '나'로 두고, 다른 글자와의 오행 생극 관계로 분류:

- 생(生): 목→화→토→금→수→목
- 극(剋): 목→토→수→화→금→목

| 관계 | 십신 | 의미 |
|---|---|---|
| 나와 같은 오행 | 비겁(比劫) | 자기 자신·경쟁·동료 |
| 내가 생하는 오행 | 식상(食傷) | 표현·재능·활동 |
| 내가 극하는 오행 | **재성(財星)** | **재물·금전** (남자는 애정운도) |
| 나를 극하는 오행 | **관성(官星)** | 직업·명예 (여자는 애정운도) |
| 나를 생하는 오행 | 인성(印星) | 문서·공부·보호 |

→ 진단 페이지의 **금전운 %** = 재성 비중 중심, **연애운 %** = 성별에 따라 재성/관성 + 화(火) 기운 가중.
(※ 재미/오픈 이벤트용 지표 — 단정·보장 표현 금지)
