# 사주 오행 리포트 템플릿

> 인쇄용 2페이지 구성. **1페이지 = 현장에서 채워서 공개**, **2페이지 = 빈칸(상담 연계용)**.
> 자이가르닉 효과(미완 → 궁금증)로 전문 상담 예약을 유도하는 것이 설계 의도.

---

## 📄 1페이지 — 나의 사주 오행 리포트 (현장 공개)

```
┌───────────────────────────────────────────────┐
│           나의 사주 오행 리포트                  │
│                                                 │
│  이름: __________   생년월일시: ______________   │
│                                                 │
│  ◆ 나의 오행 분포                                │
│     목 □   화 □   토 □   금 □   수 □   (개수)    │
│                                                 │
│  ◆ 채우면 좋은 기운(부족 오행): ____________      │
│     의미: ____________________________________   │
│                                                 │
│  ◆ 추천 색상: __________                         │
│  ◆ 추천 원석: __________                         │
│                                                 │
│  ◆ 오늘 만든 나의 팔찌 구성                       │
│     ____________________________________         │
│                                                 │
│  ─────────────────────────────────────────────  │
│  [QR] 인스타 @아이디 · 다음 예약/문의            │
└───────────────────────────────────────────────┘
```

## 📄 2페이지 — 심화 리포트 (🔒 빈칸 / 전문 상담 시 공개)

```
┌───────────────────────────────────────────────┐
│        나의 심화 운세 리포트 (전문 상담 영역)     │
│                                                 │
│  🔒 나의 10년 대운 흐름                          │
│     [ ___________________________________ ]      │
│                                                 │
│  🔒 조심해야 할 시기                             │
│     [ ___________________________________ ]      │
│                                                 │
│  🔒 운을 끌어당기는 개운법                        │
│     [ ___________________________________ ]      │
│                                                 │
│  ※ 이 영역은 전문 상담사 해독 후 채워집니다.      │
│    오늘 상담 연계를 원하시면 진행자에게 말씀해주세요.│
└───────────────────────────────────────────────┘
```

---

## 연계 멘트 (1→2페이지 전환)
> "오늘은 팔찌용 오행 분석까지 진행한 거예요. 아래 **10년 대운·개운법**은
> 저희 전문 상담사님만 해독하시는 영역이라 비워뒀어요.
> 이 빈칸 채우러 다시 예약하시는 분들이 많은데, 혹시 시간 괜찮으시면 바로 연계 도와드릴까요?"

## 하단 고정 문구 (개인정보)
> 본 리포트 작성을 위해 수집한 생년월일시·연락처는 사주 분석 및 결과 안내 목적으로만 사용되며,
> 동의하신 범위 내에서 보관·이용됩니다. (수집·이용 동의 / 광고성 정보 수신 동의 별도 체크)
