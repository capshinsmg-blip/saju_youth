## 5. 레퍼런스 체크리스트

내 계정 운영 시 참고할 수 있는 항목:

- [ ] **니치를 좁혀라**: "운세" 전체보다 "연애 상황별 타로"처럼 구체적인 하위 주제가 공감도를 높인다
- [ ] **훅 → 공감 → 행동유도(댓글)** 3단 캡션 구조를 템플릿화
- [ ] **댓글이 곧 보상(개인 결과 DM)이 되는 장치**를 만들면 좋아요보다 훨씬 높은 참여 유도 가능 (단, DM 응대 리소스 필요)
- [ ] **게시물 수보다 히트율**: 매일 올리기보다 확실히 먹히는 포맷 하나를 반복 정제하는 전략도 유효
- [ ] **짧고 정적인 릴스도 통한다**: 화려한 편집 없이 카드 이미지 + 자막만으로도 12~16초 릴스가 작동
- [ ] **참여형 퀴즈/랭킹 포맷**이 정보 전달형 콘텐츠보다 반응이 좋음 (forceteller_official 사례)
- [ ] **동일 CTA 문구 반복 삽입**(예: "댓글로 OO를 외쳐보세요")으로 댓글 지표를 관리 가능하나, 과도한 업로드 빈도(하루 3회)는 오히려 개별 게시물 반응을 희석시킬 수 있음
- [ ] 해시태그는 절제된 3~10개가 스팸성 다량 태그보다 낫다는 신호 (lifesaju.kr vs forceteller_official 비교)
