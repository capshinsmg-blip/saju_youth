# saju_youth — 사주 오행 팔찌 공방 프로젝트

"사주 오행 팔찌 공방"을 후킹(미끼)으로 사용하는 마케팅/전도 퍼널 프로젝트.

## 퍼널 한눈에 보기

```
[온라인] 인스타 메타광고 ─┐
                          ├─→ 연락처·사주정보 수집 → 공방 예약 → 방문 → 상담 연계(최종 KPI)
[오프라인] 럭키드로우 부스 ─┘
```

방문 진행: 웰컴(차) → 사주 풀이 20분 → 팔찌 제작 30~40분 → 패키징/포토존

## 핵심 수치
- 판매가: **7,000원** / 원가: **5,000원 이내**
- 진행자("잎사귀") 1명 주관 — 양성 필요
- 장소: 맬로우 스페이스 (대여)

## 🔮 3초 사주 진단 웹페이지 (`index.html`)
QR 접속용 모바일 랜딩. 검증된 만세력 라이브러리([lunar-javascript](https://github.com/6tail/lunar-javascript), 입춘·24절기 정확 적용)로 사주 8자·오행·십신을 계산해 **금전운/연애운 %** 와 **부족 오행 → 추천 색/원석**을 보여주고, 🔒 심화 리포트는 공방 방문(상담) 유도.

- **수정할 값**: `index.html` 상단 `CONFIG` — 인스타 핸들/URL, 예약폼 링크, 브랜드명 (현재 TODO 더미값)
- **로컬 미리보기**: `node preview_server.js` → http://localhost:8123
- **배포**: 정적 파일 1개라 **GitHub Pages**(Settings→Pages→main/root) 또는 Vercel/Netlify에 그대로 올리면 됨
- **계산 데이터는 기기에서만 처리** (서버 전송·저장 없음 → 개인정보 부담 적음)
